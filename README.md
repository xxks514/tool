# Hysteria 2 一键安装脚本

## 功能
- 全自动安装最新版 Hysteria 2
- 自动生成 TLS 证书
- 系统服务配置
- 支持自定义端口和密码
- 生成三种客户端配置分享格式

## 使用方法

bash <(curl -fsSL https://raw.githubusercontent.com/xxks514/tool/main/install_hysteria.sh) [端口] [密码]
